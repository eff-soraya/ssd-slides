"Passphrases, what are those?"
Similar to the example in the comic, passphrases are:

+ generally 4+ words

+ unique

+ long

+ random

+ hard for computers and people to guess

"But how do I remember all these different passphrases?"
Password Managers!
Password Managers:

+ Lots of options available, e.g. KeePass, 1Password, LastPass.

+ Encrypts your passwords.

+ Just requires one password (master password) to remember, which unlocks the others.

+ Great for saving passwords and security questions.

If you're concerned about anonymity, Tor is great because:

+ Clears search history each session.

+ Goes through a different IP address that is hard to route back to the initial user.

+ Removes browser identifiability (if you use it correctly)

+ Blocks javascript by default.

+ Has HTTPS Everywhere enabled.

+ Run by a nonprofit and dedicated volunteers.
